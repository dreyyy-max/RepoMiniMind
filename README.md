# RepoMiniMind
A mini Python chatbot that handles basic chat and simple tasks like addition.
# Overview
MiniMind is a lightweight chatbot built with Python.
The goal of this project is to practice core programming concepts, including user input handling, conditional logic and error management while gradually shaping the bot into a more interactive assistant.
